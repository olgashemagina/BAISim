35 #Inputs:
36 # map is an object of MapGeometry class or subclass of this class.
37 # data is n-by-m matrix which is the data set to fit. Each row contains
38 # m coordinates of one data point.
39 # Name can be one of the following:
40 # 'type' is one of the following strings:
41 # 'hard' is hard map with stretch = 1 and bend = 1
42 # 'medium' is more flexible map with stretch = 0.7 and bend = 0.7
43 # 'soft' is soft map with stretch = 0.5 and bend = 0.5
44 # If 'type', 'stretch' and 'bending' are omitted then 'medium' is
45 # used.
46 # 'stretch' is a positive numeric value which is the value of
47 # stretching modulo or a def with syntax
48 # val = stretch( epoch )
49 # where epoch is number of epoch (see epoch definition below) and
50 # val is the nonnegative stretching modulo to use on specified
51 # epoch. Epochs are numerated from 1.
52 # Default value corresponds to type 'medium'
53 # 'bend' is a positive numeric value which is the value of the
54 # bending modulo or a def with syntax
55 # val = bend( epoch )
56 # where epoch is number of epoch (see epoch definition below) and
57 # val is the bending modulo to use on specified epoch. Epochs are
58 # numerated from 1.
59 # Default value corresponds to type 'medium'
60 # 'weights' is n-by-1 vector of weights for data points. Weights must
61 # be nonnegative.
62 # 'intervals', intervals serves to specify user defined intervals.
63 # intervals is row vector. The first element must be zero. By
64 # default is created by usage of 'number_of_intervals' and
65 # 'intshrinkage'. Maximal value M is calculated as maximum among
66 # data points of distance from data points to the nearest node of
67 # map after initiation. Then this value is multiplied by
68 # 'intshrinkage'. All other borders are calculated as r(i) =
69 # M*i^2/p^2, where p is number_of_intervals'. Ignored if
70 # 'potential' is not specified.
71 # 'Number_of_intervals' specifies the number of intervals to
72 # automatic interval calculation. Default value is 5. Ignored if
73 # 'potential' is not specified.
74 # 'intshrinkage' is fraction of maximal distance from data points to
75 # original map which is used for intervals shrinkage (see
76 # argument delta in defineIntervals). Default value is 1 (no
77 # shrinkage). Ignored if 'potential' is not specified.
78 # 'potential' is majorant def for PQSQ. L2 distance without
79 # shrinkage is used if 'potential' is not specified.
80 #
81 # One epoch is fitting of map with fixed values of stretching and bending
82 # modulo. This process can include several iterations of two step
83 # algorithm:
84 # 1. associate each data point with nearest node.
85 # 2. recalculate node position.
86 # Process of map fitting is stopped if new values of stretching and bending

271def computeABcoefficients(intervals, potential_def_handle):
272 #PQSQR_computeABcoefficients calculates the coefficients a and b for
273 #quadratic fragments of potential def.
274 # intervals is the 1-by-K matrix of intervals' boundaries without final
275 # infinity boundary.
276 # potential_def_handle is a handle of majorant def.
277 #Get dimensions of intervals
278 p = len(intervals)
279
280 #Preallocate memory
281 A = np.zeros(p)
282 B = np.zeros(p)
283
284 #Calculate value of def all boundaries
285 pxk = potential_def_handle(intervals)
286 sxk = intervals**2
287
288 A[:p-1] = (pxk[:p-1]-pxk[1:p])/(sxk[:p-1]-sxk[1:p])
289 B[:p-1] = (pxk[1:p]*sxk[:p-1]-pxk[:p-1]*sxk[1:p]) / (sxk[:p-1]-sxk[1:p])
290 B[p-1] = pxk[p-1]
291 return A,B
